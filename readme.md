# Сайт автосалона

- Данный сайт позволяет пользователям просматривать и искать новые или поддержанные автомобили, а также позволяет производить расчет автомобиля в кредит.

- Проект создан при помощи HTML, CSS, JavaScript, Node.js, SQLite.

---

Для запуска нужно установить:
- Node.js, sqlite3, express, body-parser, path.
- Далее нужно ввести в терминал или в командную строку следующую команду : cd index
- И запустить сервер : node server.js
